Build your Properties views, no code needed

EEF is a presentation framework for the Eclipse Modeling Framework (EMF). It allows users to create rich user interfaces to edit EMF models.

Declarative

No generated code, everything is interpreted by the runtime

Simple

Your first user interface from scratch in a couple of minutes

Configurable

Customize everything easily with very powerful interpreters

EEF + Sirius

Eclipse Sirius provides a kickass bridge to create very easily EEF-based Properties views for your designers. Find more about the integration of the new EEF runtime in Eclipse Sirius in the Sirius documentation
